# Survey Form
este repositorio muestra un formulario simple que consta de ocho preguntas de las cuales:

- Tres son preguntas requeridas separadas por diferentes tipos de "Inputs" (palabras,email y contraseña).
- Hay un caja de texto pára comentario.
- Hay dos preguntas tipo **"checkbox** y **radio**" la pregunta de input tipo "**radio**" solo es de una opcion, mientras que la pregunta tipo "**checkbox**" es de opcion multiple.

ese es todo el contenido que podra apreciar en este repositorio.
